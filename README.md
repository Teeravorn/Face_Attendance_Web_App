# Face_Attendance_Web_App
Face recognition-based automated attendance system with web app

## Prerequistion
- python==3.12
- fastapi==0.135.3
- insightface==0.7.3
- numpy==2.4.4
- opencv_python==4.13.0.92
- opencv_python_headless==4.13.0.92
- pydantic==2.13.0
- uvicorn==0.44.0

### Install required library
```
pip install -r requirements.txt
```
